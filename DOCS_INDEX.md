## 🔑 Key Information

### System Requirements

- Node.js 18+ 
- Modern browser with WebAssembly support
- Internet connection (for OpenCV.js CDN)
- 500MB+ disk space

[Full requirements →](INSTALLATION.md#system-requirements)

### Supported Browsers

- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- Opera 76+
